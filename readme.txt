The amazing galery viewer from JonDesign's SmoothGallery for NextGen Gallery.

== Description ==

**Nextgen Smooth Gallery**: A Wordpress Plugin that allows you to use the amazing **JonDesign's Smooth Gallery** viewer on your **NextGen-Gallery** galleries.

In order to use the galery viewer from [JonDesign's SmoothGallery](http://smoothgallery.jondesign.net/) on your blog, you need to be using the best Wordpress gallery manager: [NextGen Gallery](http://wordpress.org/extend/plugins/nextgen-gallery/).

You can see the live demo at [Uninuni](http://uninuni.com/wordpress-plugin-nextgen-smooth-gallery/)

Just follow this simple steps to get it working

1. Go to [NextGen Gallery](http://wordpress.org/extend/plugins/nextgen-gallery/), download and activate it - Current version 0.90
1. Download our [Nextgen Smooth Gallery](http://downloads.wordpress.org/plugin/nextgen-smooth-gallery.zip)
     1. To avoid incompatibility, JonDesign's SmoothGallery 2.0 is already included on our download.
     2. Download v.010
1. Unzip it on /wp-content/plugins/
1. Activate it on your wordpress admin panel

== Installation ==

1. Go to [NextGen Gallery](http://wordpress.org/extend/plugins/nextgen-gallery/), download and activate it - Current version 0.90
1. Download [Nextgen Smooth Gallery](http://downloads.wordpress.org/plugin/nextgen-smooth-gallery.zip)
     1. To avoid incompatibility, JonDesign's SmoothGallery 2.0 is already included on our download.
     2. Download v.010
1. Unzip it on /wp-content/plugins/
1. Activate it on the admin panel - http://your_site/wp-admin/plugins.php
